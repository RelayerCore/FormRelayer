== Description ==

**FormRelayer** is the contact form plugin that small businesses actually need. No complicated setup. No developer required.

= Why FormRelayer? =

Unlike Contact Form 7, FormRelayer **stores submissions** so you never lose a lead.
Unlike WPForms, FormRelayer gives you **per-form auto-replies** in the free version.
Unlike Gravity Forms, FormRelayer is **completely free** for unlimited forms.

= Get Started in 60 Seconds =

1. Go to FormRelayer → Templates
2. Choose a template (Contact, Quote Request, Newsletter, Booking, or Feedback)
3. Add the shortcode to any page
4. Start collecting leads!

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/form-relayer/` or install through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to FormRelayer → Templates to create your first form from a template.
4. Copy the shortcode and add it to any page or post.

== Frequently Asked Questions ==

= How do I create a new form? =

Go to FormRelayer → Templates and choose from 5 pre-built templates. Your form will be created instantly with all fields configured.

= Where can I view form submissions? =

Go to FormRelayer → All Submissions. You can filter by form, status, and search by email or name.

= Can I redirect to a thank-you page after submission? =

Yes! Each form has a "Success Redirect URL" field in the Settings tab. Enter any URL and users will be redirected there instead of seeing a success message.

= Can I customize the auto-reply email per form? =

Yes! Each form has an Auto-Reply tab where you can override global settings with form-specific messages.

= Is it GDPR compliant? =

Yes! FormRelayer stores submissions in your WordPress database - you control the data. Enable the GDPR consent checkbox in Settings.

= Does it work with my theme? =

Yes! FormRelayer works with any properly coded WordPress theme and includes responsive design.

= How do I protect my forms from spam? =

FormRelayer includes honeypot protection, rate limiting, and Google reCAPTCHA v3. Configure reCAPTCHA in Settings → Privacy & Spam.

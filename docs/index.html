Introduction

FormRelayer is a modern, lightweight, and extensible form builder for WordPress. Designed for small businesses and developers alike, it bridges the gap between simplicity and power.

Fast & Lightweight

No bloat. FormRelayer loads only the assets it needs, ensuring your site stays fast.

Developer Friendly

Built with modern PHP standards, strict typing, and extensive hooks for customization.

Installation

FormRelayer follows standard WordPress plugin installation procedures.

Requirements

  • WordPress 5.8 or higher
  • PHP 7.4 or higher (PHP 8.0+ recommended)

Steps

  1. Download the form-relayer.zip file.
  2. Go to your WordPress Admin > Plugins > Add New.
  3. Click Upload Plugin and select the zip file.
  4. Activate the plugin.

Quick Start

Create your first form in under 60 seconds.

  1. Navigate to FormRelayer > Add New.
  2. Select a template (e.g., "Contact Form") or start blank.
  3. Use the drag-and-drop builder to customize fields.
  4. Click Save Form.
  5. Copy the shortcode [form-relayer id="123"] shown in the header.
  6. Paste the shortcode into any Page or Post.

Form Builder Interface

The visual builder allows you to see your form exactly as it will appear.

  • Canvas: The central area where you drag and drop fields.
  • Element Picker: Click the "+" button to add new fields like Text, Email, Checkboxes, etc.
  • Field Settings: Click any field to edit its label, placeholder, and validation rules in the sidebar.
  • Form Settings: Click the "Gear" icon to configure global form settings (Email, Confirmation, Style).

Supported Field Types

Standard Fields
  • Text / Name
  • Email / Phone
  • Textarea (Message)
  • Number
  • URL
Choice Fields
  • Select Dropdown
  • Radio Buttons
  • Checkbox
  • Hidden Field

Email Notifications

FormRelayer handles both Admin Notifications and User Auto-Replies.

Admin Notifications

Receive an email whenever a form is submitted. Configurable per form.

Auto-Replies

Send a confirmation email to the user. You can enable this in the Auto-Reply tab of Form Settings.

Email Templates

Choose from built-in templates to style your emails:

  • Default (Branded): Clean, branded email with your site logo.
  • Plain Text: Minimalist text-only.
  • Modern (Pro): Sleek, card-based design in multiple color schemes.
  • Custom HTML (Pro): Full control over HTML structure.

Integrations

Extend FormRelayer by connecting to third-party services.

Modular Architecture

Go to Settings > Integrations to configure API keys for active extensions.

PRO Features

Mailchimp Integration

Automatically subscribe form submissions to your Mailchimp Audience.

  1. Go to Settings > Integrations and enter your Mailchimp API Key.
  2. Open your Form in the Builder.
  3. Open Form Settings and enable "Marketing Integration".
  4. Enter your Audience ID and mapping fields.

Webhooks

Send submission data to any URL (Zapier, Make, Slack, Custom Scripts).

POST https://hooks.zapier.com/hooks/catch/12345/abcde

The payload will be sent as a JSON object containing all form fields.

Custom HTML Email Templates

For total control, use the Custom HTML template.

Location: Form Builder > Settings > General > Email Template > Select "Custom HTML".

A code editor will appear. You can use full HTML (<html>, <style>) and placeholders. 

<h1>New Submission</h1>
<p>You have received a new message:</p>
{all_fields}
<p>Sent from {site_name}</p>

Hooks & Filters

FormRelayer is built to be extended.

Actions

  • formrelayer_submission_complete ($submissionId, $formId, $data)
  • formrelayer_form_saved ($postId, $post)
  • formrelayer_settings_integrations_tab

Filters

  • formrelayer_email_headers ($headers, $formId)
  • formrelayer_email_subject ($subject, $formId)
  • formrelayer_spam_check ($isSpam, $data)
